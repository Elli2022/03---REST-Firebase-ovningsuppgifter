About this project

This repository gathers several small class exercises that previously lived in separate repositories. The landing page gives them one clearer home, more context, and a better entry point into each exercise.

What this lesson practices

HTTP and fetch

Working with GET, POST, PUT, PATCH, and DELETE against test APIs and Firebase.

Data shaping

Reading JSON, reshaping data, and handling different response structures.

DOM rendering

Displaying fetched data in the browser with simple vanilla JavaScript.

Firebase REST

Communicating with Firebase Realtime Database through REST-style URL endpoints.
