Connection checklist

The secure Firebase flow is in place, but this public demo still needs your actual Firebase web app details and console settings before it can read live data.

  1. Add your Firebase web app config in firebase-config.js.
  2. Enable Anonymous sign-in in Firebase Authentication.
  3. Store products under /products or update the configured path.
  4. Set Realtime Database rules so authenticated users can read products.
