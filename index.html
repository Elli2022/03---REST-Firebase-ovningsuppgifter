Om projektet

Det har repot samlar flera sma kursovningar som tidigare lag i separata repos. Landningssidan finns for att ge struktur, sammanhang och en tydligare ingang till varje delovning.

Datatolkning

Att lasa JSON, forma om data och hantera olika datastrukturer.

DOM-rendering

Att visa hamtad data i webblasaren med enkel vanilj-JavaScript.

Firebase REST

Att kommunicera med Firebase Realtime Database via URL-endpoints.
